
# IP Scanner

A simple IP Scanner which tells what ip address are up in your network 


## Installation

Install my-project with npm

```bash
  git clone "https://github.com/webdeveloperAryan/ipscanner"
  cd ipscanner
  chmod +x 'filename'
  ./filename "ip"
```
